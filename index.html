Skipify

Quickly skip songs in Spotify (Mac) using a keystroke

View project on GitHub

What's Skipfiy?

Simply put, Skipify allows you to assign a hotkey to skip to the next song in Spotify. It can be used even when Spotify is in the background (ex: I'm browsing in Chrome and too lazy to switch to Spotify and click a button - I know this isn't just me).

Setup

  1. Download Skipify by clicking on the blue button that says Download Binary in small lettering on top. It should have parentheses and a version number with it in the text below, for instance ".zip file (1.0)".
  2. Double click the Skipify file you just unzipped from the download to open it up and install it as a "Service".

  3. Go to System Preferences > Keyboard > Shortcuts.

  4. Select Services from the left column.

  5. Scroll down to the bottom of the list, under the General tab.

  6. Click the add shortcut button next to the line with Skipify.

  7. Hold down your preferred shortcut. I use CTRL-CMD-], since that's what I use to go forward a page when in Chrome. Make sure the shortcut isn't used by another application first!

  8. That's it! You can make sure it's running by going in the toolbar to [Application] > Services and it should say Skipify with the blue icon next to it.

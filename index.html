Skipify

Quickly skip songs in Spotify (Mac) using a keystroke

View project on GitHub

What's Skipfiy?

Simply put, Skipify allows you to assign a hotkey to skip to the next song in Spotify. It can be used even when Spotify is in the background (ex: I'm browsing in Chrome and too lazy to switch to Spotify and click a button - I know this isn't just me).

Setup

  1. Double click the Skipify file you just unzipped from the download to open it up and install it as a "Service". --screenshot here--

  2. Go to System Preferences > Keyboard > Shortcuts. --screenshot here--

  3. Select Services from the left column. --screenshot here--

  4. Scroll down to the bottom of the list, under the General tab.

  5. Click the add shortcut button next to the line with Skipify.

  6. Hold down your preferred shortcut. I use CTRL-CMD-], since that's what I use to go forward a page when in Chrome. Make sure the shortcut isn't used by another application first!

  7. That's it! You can make sure it's running by going in the toolbar to [Application] > Services and it should say Skipify with the blue icon next to it.
